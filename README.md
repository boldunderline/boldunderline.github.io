# Lucky Wheel / Wheel of Names 

Lucky Wheel / Wheel of Names used for Temu Responden BI Sibolga 2022 event

Features:
- Upload data from `.xlx`, `.xlsx`, or `.csv` (data saved in `localStorage`)
- Show list of participants and list of winners 
- Button to reset winners or delete all data

> Code based on [lucky-wheel](https://github.com/HellCatVN/lucky-wheel) by [HellCatVn](https://github.com/HellCatVN)
